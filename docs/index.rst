Welcome to BugDoc's documentation!
==================================

`BugDoc` is a framework for finding root causes of errors in computational pipelines.
For more detailed information about the framework, please refer to our SIGMOD paper:

`BugDoc: Algorithms to Debug Computational Processes. Raoni Louren√ßo, Juliana Freire, and Dennis Shasha. In ACM SIGMOD, 2020. <https://arxiv.org/abs/2004.06530>`_

You can find the source code on GitHub: https://github.com/VIDA-NYU/BugDoc
